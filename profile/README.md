# AirBuddy — Portable Wireless Device Battery Monitoring

## About AirBuddy

AirBuddy provides a complete battery monitoring environment for users who want to track their wireless devices. AirBuddy combines battery levels for AirPods, keyboards, mice, trackpads, and headphones with popup notifications. AirBuddy enables users to see battery percentages when devices connect, disconnect, or run low. AirBuddy runs on desktop platforms. AirBuddy is trusted by AirPods owners, wireless keyboard/mouse users, and anyone with Bluetooth peripherals.
